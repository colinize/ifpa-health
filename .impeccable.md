## Design Context

### Users
Just the creator — for now. A smart, data-literate pinball insider checking the scene's pulse at a glance. One person, one screen, one question: "is competitive pinball growing or dying?"

### Brand Personality
**Smart, nerdy, accurate, refined.** An editor with a spreadsheet open. Numbers first, confident tone, zero fluff. Authoritative without shouting. On-brand with Kineticist — same voice, quieter register.

### Aesthetic Direction

**Visual tone:** Data tool with editorial gravity. A magazine stats page, not a SaaS dashboard. Should feel like a Kineticist article that happens to be entirely numbers.

**Relationship to Kineticist:** Full inheritance. Same surface, same type system, same palette, same "no cards, no chrome" rule. This is a Kineticist side-project, so it should look like it belongs on the same shelf.

**Palette:**
- **Surface:** Kineticist straw `#fffce8` — the base background, not white
- **Text:** Charcoal scale (`charcoal-25` → `charcoal-800`). Pure black `#0a0c0d` reserved for headlines and primary numbers
- **Accents:** Pink `#f15bb5`, teal `#00f5d4`, yellow `#fee440` for states, highlights, and trend indicators
- **Functional tokens:** `--up` / `--down` / `--flat` map to teal / pink / charcoal-400 (or similar) for trend direction
- **White:** Reserved for data callouts and the gauge fill — never as a container background

**Typography:** Source Serif 4 for the narrative sentence and any long-form explainer copy. DM Sans for numbers, labels, nav, everything UI. Same rule as Kineticist: serif tells, sans shows. Loaded via `next/font/google`.

**Surfaces:** No cards. Dividers and whitespace only. Answer cards are sections, not boxes. White fills reserved for callouts, never as containers.

**Layout:** Single viewport on desktop. 8px grid. Detail drawer is opt-in, not default.

**Motion:** Count-up on the gauge is allowed — the one moment of delight. Everything else businesslike, 80-200ms. `prefers-reduced-motion` kills motion entirely.

**Icons:** Lucide, 24px grid, 1.5px stroke. Outline default.

**References:** FiveThirtyEight (data editorial), The Athletic league dashboards, Defector, Pitchfork scores, Bloomberg Terminal (density + intentionality).

**Anti-references (all AI slop tells):**
- Dark-mode dashboard with neon accents (the current look — that's default Claude taste, starting fresh)
- Purple/blue/cyan gradient hero
- Gradient text
- Glassmorphism, frosted cards, blurred panels
- Identical card grids with identical icons
- Hero metric + three equal-sized stat cards pattern
- Generic Inter / system-ui-only typography
- Shadcn-default appearance (neutral gray cards, `bg-card`, drop shadows, rounded-2xl everywhere)
- Emoji icons used as UI chrome
- AI dashboard template: centered headline + KPI row + line chart + table
- Retro arcade nostalgia (pixel fonts, CRT glow)
- Corporate SaaS blue
- Recharts default style
- Faux-serif display fonts used ironically

### Accessibility
- WCAG AA contrast on charcoal text over straw (verify `charcoal-400` and lighter against `#fffce8`)
- `prefers-reduced-motion`: gauge count-up and transitions become instant
- Semantic HTML first (native `<details>`/`<summary>` already in use — keep that pattern)
- ARIA labels on gauges
- Focus: `2px solid #2d62ff` with 2px offset, matching Kineticist
- Theme toggle: drop it. Light only, matching Kineticist. Dark mode deferred.

### Design Principles

1. **Numbers are the product.** Typography, spacing, and contrast serve legibility of numbers first. If a chrome element competes with a number, remove the chrome.

2. **Serif tells, sans shows.** Same Kineticist rule. Narrative sentence in serif; gauge readout, card stats, labels, nav in sans.

3. **No cards, no chrome.** Sections are divided by whitespace and hairline rules, not containers with rounded corners and shadows.

4. **One moment of delight.** The gauge count-up. Everything else is quiet. Don't dilute.

5. **Look nothing like an AI dashboard.** When a decision is ambiguous, pick the choice an AI wouldn't. Editorial layout over symmetric grids. Confident typography with personality over neutral Inter. Straw surface over gray cards. Numbers set at magazine sizes, not crammed into equal-sized boxes.

6. **Start from zero.** The current implementation is default Claude taste (dark-first oklch, shadcn primitives, neutral cards). Treat it as scaffolding, not a baseline — nothing in the current visual layer is sacred. Layout, tokens, components, and motion all up for rework.

### Stack
Next.js 16 + React 19, Tailwind v4 (oklch tokens), Supabase, Vercel. Server Components by default. Client components only for animation (gauge count-up) and stateful UI (theme toggle, detail drawer).
